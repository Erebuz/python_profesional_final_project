# Проектная работа для курса Python Professional

## Описание

Проект состоит из 3 частей:

- backend - сервер, принимающий исходный поток (RTSP, веб-камера, видеофайл). Сервер реализует обработку потока с помощью нейросети, а также веб-сервер.
- mmtx - сервер MediaMTX, принимает поток от backend, и реализует передачу в RTSP, HLS.
- frontend - небольшой фронтенд для демонстрации работы проекта.
