# Проектная работа для курса Python Professional

## Идея проекта

Проект представляет собой сервер, обрабатывающий входящий видео поток (RTSP, камера, видеофайл) с помощью нейросети, и выдающий результат в виде RTSP и HLS потоков. 

Кроме того, в проекте реализован веб-интерфейс для управления настройками разпознавания нейросетью объектов.

## Описание

Проект состоит из 3 частей:

- backend - сервер, принимающий исходный поток. Сервер реализует обработку потока с помощью нейросети, а также веб-сервер.
- mmtx - сервер MediaMTX, принимает поток от backend, и реализует передачу в RTSP, HLS.
- frontend - небольшой фронтенд для демонстрации работы проекта.

## Установка

В сборке настроен файл docker-compose.yml для локальной сборки и запуска.

Кроме того, проект собран в github actions в 3 docker образа, который можно напрямую скачать и запустить так же с помощью docker-compose.
